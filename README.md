# yl-skills

引领 (yinling) 前端项目 Claude Code Skills 集合。

## 技能列表

| 技能 | 说明 |
|------|------|
| `yl-frontend-pc-project-shared` | 引领 PC 前端跨版本通用规范（API 格式、权限理念、枚举、工具函数、部署） |
| `yl-frontend-pc-project-vue3` | 引领 Vue 3 PC 端开发规范（Composition API + Vite + Element Plus + Pinia） |
| `yl-frontend-pc-project-vue2` | 引领 Vue 2 PC 端开发规范（Options API + Vue CLI + Element UI + Vuex） |
| `yl-frontend-uniapp-project` | 引领 uni-app 移动端开发规范（uni-app + Vue 3 + Pinia + my-* 组件） |

## 使用方法

在 CC Switch 或 OpenCode 中添加自定义仓库：

| 字段 | 值 |
|------|-----|
| Owner | fengmazi |
| Name | yl-skills |
| Branch | main |
